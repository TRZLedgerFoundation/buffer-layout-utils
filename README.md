# `@trezoa/buffer-layout-utils`

A TypeScript/JavaScript library providing utilities for working with binary data structures using buffer layouts in the Trezoa ecosystem.

## Installation

```bash
npm install @trezoa/buffer-layout-utils
# or
yarn add @trezoa/buffer-layout-utils
# or
pnpm add @trezoa/buffer-layout-utils
```

## Overview

This package provides utility functions and layouts for working with binary data serialization and deserialization, particularly useful for blockchain applications and data structures that require precise byte-level control.

## Features

- **BigInt Support**: Utilities for handling big integers in buffer layouts
- **Decimal Handling**: Tools for working with decimal numbers in binary format
- **Native Types**: Support for native data types and structures
- **Web3 Integration**: Seamless integration with Trezoa Web3 functionality
- **TypeScript Support**: Full TypeScript definitions included
